🎯 Emergency Flow
User triggers emergency → Voice/Manual detection → Backend processes alert → Real-time broadcast to contacts and nearby users → Location shared → Recording started
🎤 Voice Recognition
Browser Speech API captures audio → React component processes transcript → Matches safety phrase → Triggers emergency via WebSocket → Backend validates and broadcasts
📍 Location Tracking
HTML5 Geolocation API → Real-time coordinates → Backend calculates nearby users → Location history stored → Emergency services lookup
⚡ Real-time Communication
WebSocket connection maintains live link → Emergency alerts broadcast instantly → Location updates stream continuously → Voice events trigger immediately
